# 🚀 JanMitra AI

### Your Voice. Your City.

JanMitra AI is an AI-powered civic issue reporting platform that enables citizens to report local problems and allows authorities to resolve them efficiently through a real-time admin dashboard.

Built as part of the **30 Days • 30 SIH Solutions Challenge**.

---

## 📱 Features

### Citizen App

* 📸 Photo-based complaint reporting
* 🎤 Voice complaint submission
* 🤖 Gemini AI complaint analysis
* 📍 Current location detection
* 🔔 Real-time notifications
* 📋 Complaint tracking
* 👍 Community verification system
* 🌐 Multi-language support

---

### Admin Dashboard

* 📊 Real-time analytics

* 📋 Complaint management

## 🚀 Installation

```bash
git clone https://github.com/yourusername/janmitra-ai.git

cd janmitra-ai
```

Configure Firebase:

```javascript
firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_BUCKET",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

Add your Gemini API key.

Run locally:

```bash
Open index.html
```
